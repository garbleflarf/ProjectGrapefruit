# Project Grapefruit

A Jellyfin frontend for the New Nintendo 3DS!

A sister project to [Project Cappuccino](https://github.com/dogeeeeeeeeeeeee/ProjectCappuccino).

# Installation

Installation is simple.
Download the repository and extract it to your htdocs folder.
You must have Jellyfin, first of all, a webserver, and PHP installed for this to work.
